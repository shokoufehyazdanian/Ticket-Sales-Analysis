
#  Ticket Sales & Revenue Analysis

This project analyzes ticket sales data to uncover insights about **customer purchase behavior**, **lead time (days in advance)**, and **revenue trends over time**.  
It also explores **early-bird opportunities** and includes a **churn prediction proof-of-concept**.

##  Key Features
- Calculate lead time (days before ticket usage)
- Compare lead times by product type and customer group
- Analyze daily and monthly sales & revenue trends
- Identify seasonality and demand patterns
- Provide data-driven recommendations for early-bird discounts
- (Optional) Churn prediction using logistic regression
